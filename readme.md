# 🧠 AI Code Debugger Assistance

An intelligent **AI-powered debugging tool** that analyzes code errors from screenshots and provides **clear explanations, hints, and fully corrected solutions**.

🚀 Built to simulate a real-world developer assistant using **Streamlit + Gemini API**

## 📌 Key Features

* 📤 Upload code error screenshots (PNG, JPG, JPEG)
* 🧠 AI analyzes code directly from images
* 🚨 Clear and beginner-friendly **error explanations**
* 💡 Step-by-step **hints for learning**
* ✅ **Fully corrected code solutions**
* ⚙️ Two modes:

  * **Hints Mode** → guided debugging
  * **Solution Mode** → complete fix
* ⏳ Smooth UX with loading indicators (spinners)
* 🎯 Clean and professional UI (Streamlit)

---

## 🛠️ Tech Stack

* **Frontend/UI:** Streamlit
* **AI Integration:** Gemini API (Google Generative AI)
* **Image Processing:** PIL (Python Imaging Library)
* **Environment Management:** Python venv + dotenv

---

## 🧠 How It Works

1. Upload a screenshot containing code/error
2. Select debugging mode (Hints / Solution)
3. Click **Debug Code**
4. AI:

   * Extracts and understands the code
   * Detects the issue
   * Explains the error
   * Provides fixes or full solution

## ⚡ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Dev-Maidul/AI-Code-Debugger-App.git

# Navigate to project
cd AI-Code-Debugger-App

# Create virtual environment
python3 -m venv codedebugger
source codedebugger/bin/activate   # Mac/Linux

# Install dependencies
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
GEMINI_API_KEY=your_api_key_here
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```
